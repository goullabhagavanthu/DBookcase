D!Bookcase

Description:
 A platform designed to help you sell books from my personal collection, offering a seamless way to list and manage my inventory. Empower your book-selling journey with a professional, user-friendly marketplace tailored to your needs.

Installation
Clone the repository:

git clone <https://github.com/goullabhagavanthu/DBookcase>
Navigate to the project directory:

cd <DBook>
(Optional) Create and activate a virtual environment:

python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
Install dependencies:

pip install -r requirements.txt
Usage
Run the application:
python project.py
Open a web browser and go to: http://127.0.0.1:8000/

Follow the on-screen instructions or interact with the application as designed.

Testing
Run the tests using pytest:


pytest test_project.py
